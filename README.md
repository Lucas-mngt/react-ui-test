### Installing

First clone project and install dependencies

```sh
$ mkdir react-ui-test && cd react-ui-test
$ git clone https://github.com/Lucas-mngt/react-ui-test.git
$ npm install
```

Run on this command on terminal to run the server Node

```sh
$ node server
```

Navigate to [http://localhost:5000](http://localhost:5000)

Run on this command on terminal to run the UI

```sh
$ npm run client
```

Or in the client folder

```sh
$ npm start
```

Navigate to [http://localhost:3000](http://localhost:3000)

## Built With

- [React](https://reactjs.org/)
